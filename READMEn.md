# EliteShirts

## Descripción

Sistema de gestión de ventas para una tienda de camisas desarrollado con ASP.NET Core, SQL Server y Vue.js.

## Tecnologías

* ASP.NET Core 8
* Entity Framework Core
* SQL Server
* JWT Authentication
* Vue 3
* Axios
* Bootstrap

## Funcionalidades

* CRUD de Productos
* CRUD de Clientes
* CRUD de Ventas
* Login con JWT
* Arquitectura en Capas
* SQL Server

## Ejecución Backend

1. Abrir EliteShirts.API
2. Configurar la cadena de conexión
3. Ejecutar las migraciones
4. Ejecutar la API

## Ejecución Frontend

1. Entrar a la carpeta Fronted
2. Ejecutar:

npm install

npm run dev
